About Cryo Packet Sniffer

Description

This project is a minimal packet sniffer inspired by Wireshark, built with Python, Scapy, and Flask. It captures network packets, displays key details, and identifies if packets are encrypted or potentially vulnerable.

Technologies Used

  • Python
  • Scapy
  • Flask
  • Tabulate
  • HTML/CSS/JS

How it Works

The sniffer captures packets on the Wi-Fi interface for 10 seconds, analyzes them, and displays results in real time. It highlights whether packets are encrypted (e.g., HTTPS) or vulnerable (e.g., HTTP).
